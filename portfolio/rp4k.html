Real Programming 4 Kids

Real Programming is a tech education startup company with locations across Canada. The company focusses on video game programming/design as a way of teaching young learners more sophisticated coding practices.


My Work as a Video Game Programming Instructor

Utilizing a variety of software tools and programming languages, I currently lead small classes of students through object-oriented programming lessons, gradually building a working video game from beginning to end. Along the way, time is taken to explain fundamental coding concepts and practices - such as refactoring, version control, and breakpoints.

In addition, the mathematics widely used to program the movement of sprites and other game objects - especially trigonometry - is explained. While this can sometimes lead to some bored students, it makes for better coders!

Here are some of the tools I used to build video games and teach lessons:

  • Programming languages:
    • Java 6
    • Python 3
    • Visual Basic
    • C++
    • JavaScript
  • Tools (IDEs and Game Engines):
    • Microsoft Visual Studio 2017
    • Netbeans Java IDE
    • Unity 5.5.4
    • Alice 2.4
    • PyCharm
